## aws cloudwatch, 
- buat monitor performa resource nya aws, misal servicenya aws ec2 itu pake cpu nya berapa, memorinya berapa 

setup webserver apache: 

sudo systemctl status httpd.service

sudo systemctl start httpd.service

sudo systemctl enable httpd

sudo systemctl status httpd.service

expected output: 
httpd.service - The Apache HTTP Server
   Loaded: loaded (/usr/lib/systemd/system/httpd.service; enabled; vendor preset: disabled)
   Active: active (running) since Wed 2025-10-22 13:13:57 UTC; 6min ago

 Main PID: 2577 (httpd)
   Status: "Total requests: 2; Idle/Busy workers 100/0;Requests/sec: 0.00514; Bytes served/sec:   1 B/sec"
  


di aws cloudwatch, monitoring itu setiap 5 menit. Tapi klo di linuxnya (htop) itu realtime.
